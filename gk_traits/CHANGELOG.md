Added the full species trait suite: 57 traits (37 positive, 20 negative) across nine files in common/traits/ split by gate and polarity - global, common, phenotype, and cyber/biogenesis, plus special. Shared gate blocks live in common/inline_scripts/traits/ (portrait lists per overlay category, genemod access per polarity, species_class_override per gate). CYBER-leg traits reach MACHINE and ROBOT archetypes via the is_archetype branch from trait_pc_ark_preference, and reach ascended cyborgs regardless of portrait via a species_class_override on has_trait = trait_cybernetic. Phenotype-leg traits carry the Mutation-ascension override so gene modding can apply them across phenotypes; biogenesis-leg traits instead key on the Engineered Evolution ascension perk, which is the single perk behind all four genetics trees (base, cloning, purity, mutation). Replaces the Tinkerer placeholder (now trait_gk_tinkerers) and the three marker traits, whose portrait lists became the inline gate scripts.
Added gk_diet_upkeep_mult static modifier in common/static_modifiers/: one key fanning out to the three organics upkeep modifiers with a single consolidated tooltip, used by five traits. Added common/script_values/gk_script_values.txt with counter_gk_cannibalistic (clone of counter_familial, planet-scoped, capped at 20000) and gk_aeroplankton_size_mult.
Last Children scales -0.5% workforce per 100 own-trait pops empire-wide and uncapped, against a flat +30%, netting roughly +5% at game start. Pop counts are raw, so per-100 rates encode as the rate divided by 100 - the same encoding trait_familial uses; Cannibalistic's -0.1%/100 follows the same form.
Added Burnout Susceptibility infrastructure: gk_monthly_burnout_chance scripted modifier, a monthly country pulse in common/on_actions/, and the weighted leader death roll in events/gk_burnout_events.txt. Research payout on death is not yet implemented.
Added tools/steal_icons.py, which resolves each shipping trait's source-mod icon and copies it in renamed to the gk trait id (47 of 57; 10 authored traits await art).
Widened vanilla phenotype-gated traits via whole-file overwrites in common/traits/ for 04_species_traits.txt (9), 09_ascension_traits.txt (1), 15_biogenesis_species_traits.txt (16), 16_infernals_traits.txt (4), 17_shroud_species_traits.txt (4). Each trait's portrait_override is the union of vanilla entries and cross-tagged portraits from the sheet; vanilla DLC triggers and single-portrait exceptions preserved.
Added portrait cross-listing for 12 portraits flagged in the sheet's notes (humanoid_hp_01, rep17, fun13, tox9, lith6, lith8, inf1, inf2, inf3, inf6, inf7, inf10). New portrait_sets in common/portrait_sets/01_gk_crosslist_sets.txt bundle portraits per target phenotype + DLC-gate; common/portrait_categories/zz_gk_categories.txt extends the relevant UI dropdowns. All cross-listings DLC-gated so non-DLC players don't see them.
Etched Frames and Pestilent Byproducts now display their output. resources/produces blocks never auto-generate trait tooltip text, so both got hand-written trait_<id>_effect keys, following the vanilla TRAIT_DRAKE_SCALED_EFFECT and inorganic_breath_tooltip phrasing.
Every gk trait now lists the correct categories. localized_tags was missing from all of gk_traits_cyberbio_positive.txt and from Burnout Susceptibility, and Hypermetabolism, Fire Breath, Ferroelectric, and Elders had each dropped one of their species_class legs. Added two mod-side categories for the overlay legs, since no vanilla key covers them: SPECIES_TRAIT_GK_CYBER points at concept_gk_mechanist_traits, named Mechanist because these portraits are not mechanical yet but their evolution mechanics push them there (SPECIES_TRAIT_CYBORG names only cybernetic traits, while our cyber leg also reaches synthetic and machine portraits and the MACHINE and ROBOT archetypes), and SPECIES_TRAIT_GK_BIOGENESIS points at concept_gk_biogenesis_traits. Both concepts open with their access rules and then list every trait in the suite, matching how vanilla presents Avian Species Traits and the other phenotype categories. Ship-set-only anchor classes (PSIONIC, BIOGENESIS_02) no longer emit a phenotype category, since they are never-match anchors rather than real phenotype gates.
Etched Frames now starts at two ascension perks' worth of unity rather than nothing, via gk_etched_frames_scaling (base 2, plus one per perk taken). Yields 0.10 unity per 100 pops at game start, rising 0.05 per perk.
Trait category tooltips now list every gk trait. The per-phenotype lists in those popups are hand-written enumerations in vanilla localisation rather than anything the engine derives from localized_tags, so all 40 gk traits carrying a phenotype tag were absent from them. Added localisation/replace/english/gk_phenotype_concepts_l_english.yml, which rebuilds the twelve vanilla phenotype lists with our traits appended and adds a psionic list. Uses the replace directory rather than a plain loc file so the override holds regardless of load order. Also picks up three vanilla omissions: Bloomed was missing from both the plantoid and fungoid lists, and Infernal from the infernal list, despite all three carrying the tag.
Extrasensory, Death Echoes, Panoptics, and Burnout Susceptibility now appear under Psionic. Each listed PSIONIC in species_class but omitted SPECIES_TRAIT_PSIONIC from localized_tags, so they were takeable by psionic species while being advertised nowhere. PSIONIC is the one class the overlay inline scripts do not supply, so it has to be written on both lines by hand. Swept all 40 gk traits for the same drift; these four were the only cases.
Widened vanilla traits no longer warn on load. A trait carrying portrait_override needs a matching _portrait_override_tt key to explain the restriction in its tooltip, and vanilla only ships one for the traits it restricts itself, so the 18 traits we widened that vanilla left open were logging errors and dropping the explanation line. Added keys for all of them, grouped so five strings cover fifteen traits by substitution.
Biogenesis-leg traits now unlock from the Engineered Evolution ascension perk instead of the Unnatural Selection tradition, reaching all four genetics trees rather than mutation alone, via a new gk_class_override_biogen inline script. The cyber leg drops its Unnatural Selection branch entirely and now keys only on has_trait = trait_cybernetic, so mutation empires no longer reach Mechanist traits without cyberising. Phenotype-leg traits keep the original tradition gate, which is why the shared override split into three files rather than being edited in place.
