# membership is pure UI grouping and the SAME set may be listed under several
# categories - see zz_gk_categories.txt. Vanilla precedent: cybernetic_avians is
# species_class = AVI and appears under both `avians` and `cybernetics`.
#
# Portraits with two phenotypes appear in a set for each, so they show in both
# dropdowns. No DLC conditional_portraits gate: these are mod assets, not DLC
# assets. Generated from PORTRAIT_MAPPING.csv; keep in step with it.
